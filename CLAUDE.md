## 1. 서비스 개요

- **서비스명:** 숨표 (SOOM)
- **한 줄 소개:** 바쁜 일상 속, 나만의 조용한 쉼을 발견하는 감성 휴식 스팟 공유 플랫폼
- **문제:** 현대 사회인은 조용하고 감성적인 휴식 공간을 신뢰성 있게 찾기 어렵다. 기존 서비스는 광고성 후기와 유명 장소 위주라 실제로 편히 쉴 곳을 찾는 데 시간·피로가 든다.
- **타깃:** 스트레스를 느끼는 20~40대 직장인·대학생 / 혼자 조용히 쉬거나 산책하는 시간을 중요하게 여기는 사용자 / 숨겨진 힐링 스팟을 찾고 싶은 사람
- **대표 페르소나:** 김서연(29세, 마케팅 직장인). 실제 사용자 경험 기반의 신뢰할 수 있는 힐링 장소 정보를 원함.

---

## 2. 기술 스택

- **런타임/프레임워크:** React Native + Expo (SDK 최신 안정 버전)
- **언어:** TypeScript
- **네비게이션:** Expo Router (파일 기반 라우팅) 또는 React Navigation
- **상태관리:** 초기에는 React state / Context. 필요 시 Zustand 도입 고려
- **지도:** react-native-maps (Google/Apple Maps)
- **스타일:** StyleSheet 기반 + 아래 디자인 토큰을 상수로 관리

> 아직 초기 세팅 전이라면 `npx create-expo-app` 기반으로 시작합니다.
> 새 라이브러리를 추가하기 전에는 반드시 먼저 제안하고 확인받습니다.

## 3. 디자인 시스템

차분하고 따뜻한 세이지 그린 + 베이지 계열. 여백이 넉넉하고 편안한 감성 모바일 UI.

### 컬러 토큰
| 이름 | 값(예시) | 용도 |
|---|---|---|
| `sage` | `#8DA284` | 메인 브랜드/버튼/활성 태그 |
| `sageLight` | `#A8B89E` | 보조 강조 |
| `beigeBg` | `#F2EEE6` | 앱 배경 |
| `cardBg` | `#FFFFFF` | 카드/입력 영역 |
| `textMain` | `#4A4A46` | 본문 텍스트 |
| `textSub` | `#9A9A93` | 보조/플레이스홀더 |
| `star` | `#F5C24B` | 별점 |
| `logout` | `#E8837A` | 로그아웃 등 경고성 액션 |

> 위 값은 시안 기반 추정치입니다. 확정 색상표가 생기면 이 표를 우선합니다.

### 스타일 원칙
- 라운드 코너: 카드·버튼 16~24px, 태그 칩 full-round
- 넉넉한 여백(패딩 16~24px), 톤 다운된 그림자
- 태그 칩: 선택 시 sage 배경 + 흰 글씨, 미선택 시 흰 배경 + 회색 글씨
- 혼잡도 배지: 여유로움(초록) / 보통(주황) / 조금 붐빔(빨강)

## 5. 핵심 기능

1. 감성 휴식 스팟 탐색 (위치 기반 추천)
2. 사용자 후기 및 사진 공유
3. 분위기 태그 시스템 (조용함·야경·산책·혼자 가기 좋음·힐링)
4. 지도 기반 탐색
5. 저장(북마크) 기능
6. 혼잡도 및 시간대 정보
7. 맞춤 추천 기능

---

## 6. 대표 User Flow

**목표: 힐링 스팟을 탐색하고 후기 등록하기**

앱 실행 → 로그인 확인 → 홈에서 탐색 진입 → 태그 필터("조용함") → 지도 마커 클릭 → 장소 상세 확인 → 후기 목록 스크롤 → 후기 작성(사진·태그·텍스트) → 등록 완료(상세로 복귀) → 북마크 저장

## 8. 코딩 규칙

- 화면 단위로 하나씩 구현. 한 번에 전체를 만들지 않는다.
- 컴포넌트는 재사용 가능하게(SpotCard, TagChip, CongestionBadge, StarRating 등).
- 디자인 토큰은 하드코딩하지 말고 `constants/theme.ts` 같은 곳에서 import.
- 커밋 전 실제로 실행 가능한 상태인지 확인.
- 불확실하면 추측하지 말고 먼저 질문한다.
